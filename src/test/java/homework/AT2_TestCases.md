# AT-2. Тест-кейсы для формы расчёта стоимости билета в кино

## TC-1 Корректный расчёт без скидок
Шаги:
1. Возраст: 25
2. Дата: будний день
3. Время: 19:00
4. Фильм: Криминальное чтиво - R (18+)
   Ожидаемый результат:
- Стоимость билета 500 рублей

## TC-2 Детская скидка
Шаги:
Возраст: 5
1. Дата: будний день
2. Время: 16:00
3. Фильм: Король-лев - G (0+)
4. Нажать «Рассчитать»

Ожидаемый результат:
Стоимость билета: 500 − 250 = 250 рублей

## TC-3. Несоответствие возрастного ограничения
Шаги:
1. Возраст: 16
2. Дата: любой доступный день
3. Время: 19:00
4. Фильм: Криминальное чтиво - R (18+)
5. Нажать «Рассчитать»

Ожидаемый результат:
Выводится сообщение о том, что возраст не подходит для просмотра фильма. //подправить

## TC-5. Проверка суммирования всех скидок
Шаги:
1. Возраст: 5
2. Дата: будний день
3. Время: 09:30
4. Фильм: Король-лев - G (0+)
5. Нажать «Рассчитать»

Расчёт:
- Базовая цена: 500
- Детская скидка: −250
- Будний день: −100
- Утренний сеанс: −50

Ожидаемый результат:
Стоимость билета: 100 рублей
